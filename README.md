## ğŸ¨ Customization

### Changing Images
Replace Unsplash URLs in `index.html`:
```html
<div class="product-image" style="background-image: url('YOUR_IMAGE_URL')"></div>
```

### Modifying Colors
Update CSS variables in `style.css`:
```css
:root {
  --primary-color: #f0a500;
  --secondary-color: #333;
}
```

### Adjusting Parallax Speed
Modify the speed value in `script.js`:
```javascript
const speed = 0.5; // Adjust between 0 and 1
```
